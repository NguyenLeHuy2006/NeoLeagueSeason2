VisionBin automatic sorting system.

Using ESP32-CAM and AI recognition, the system identifies the item in the user's hand. The system will automatically open the corresponding compartment: Organic, Recyclable, Non-Recyclable, or Hazardous..

Core Components & Hardware.

ESP32-CAM

Image recognition using AI.

Sensors

PIR and ultrasonic sensors measure the level of debris.

Servo Motors

Control the trash can door.

Python & C++

Operating systems and data processing.

Deployment & Scalability.

  • Extremely low cost: Only $25 per trash can (20 times cheaper than industrial-grade).
  • Scalability: Deployment in schools, offices, and shopping malls.
  • Energy saving: Deep Sleep mode helps optimize the 18650 battery.

Smart Waste Monitoring Application.
